# Hutchinson, KS voter turnout

Datafiles, downloaded from [National Neighborhood Data Archive (NaNDA)](https://www.icpsr.umich.edu/sites/nanda/view/studies/38506) at the University of Michigan amnd R code are in the folders above.

Non-Presidential Years :
* National Median Turnout:  0.4311 
* Hutchinson Turnout:  0.4242 
* Hutchinson Median Percentile:  40.8 %

Presidential Years :
* National Median Turnout:  0.6041 
* Hutchinson Turnout:  0.5555 
* Hutchinson Median Percentile:  30.2 %

[Yearly summary](https://github.com/badgettrg/Hutchinson-KS-voter-turnout/blob/main/data/National_and_Hutch_voter_turnout_since_2004.csv)
